## 技術棧

- **前端框架**: Next.js 14
- **樣式**: TailwindCSS
- **字體**: Playfair Display, 中文字體
- **圖標**: Font Awesome
- **構建工具**: pnpm

## 開發說明

```bash
# 安裝依賴
pnpm install

# 開發模式
pnpm dev

# 構建生產版本
pnpm build

# 啟動生產服務器
pnpm start
```
